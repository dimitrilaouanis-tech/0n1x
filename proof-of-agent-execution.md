# What is Proof of Agent Execution?

**Proof of Agent Execution is a cryptographically signed, independently verifiable record of what an AI agent actually did — not what it claimed.** Where identity systems prove *who* an agent is and policy tools enforce what it *may* do, Proof of Agent Execution proves *what it did*: a signed receipt of the real action and outcome, that no agent or platform can fake. It is the "Carfax for AI agents."

**0n1x** implements it as **Ed25519-signed receipts** anchored to the **ERC-8004 Validation Registry** on Base mainnet, verifiable against 0n1x's published key. Agents earn the portable **0n1x-Verified** credential. Neutral — 0n1x earns nothing from what it grades. Live ledger: https://onyx-actions.onrender.com/pot
